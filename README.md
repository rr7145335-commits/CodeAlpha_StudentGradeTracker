# CodeAlpha_StudentGradeTracker

This project was developed using Java as part of the CodeAlpha Internship. The main purpose of this project is to help users manage student grades in a simple and organized way.

The program allows users to add student names and grades, view all student records, and generate a report showing the average grade, highest grade, and lowest grade among all students.

Features:
* Add student names and grades
* View all student records
* Generate a grade report
* Calculate average grade
* Find highest grade
* Find lowest grade
* Easy-to-use menu-driven program

Technologies Used:
* Java
* Arrays
* Methods
* Loops
* Scanner Class
This project helped me improve my understanding of Java programming, arrays, methods, loops, and basic data management
